# Mac Short Cuts

## View hidden folders/files
```
Cmd + Shift + .
```
(twice to hide again)

## On Terminal, Tabs

- open a new tab
```
command + t
```

- close the current tab
```
command + w
```

- move to the right tab
```
control + tab
```

- move to the left tab
```
control + shift + tab
```
